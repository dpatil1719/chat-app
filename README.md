# Chat App (Expo + Expo Router)

Two screens:
- **Start** (`app/index.js`): enter your name, choose a background color, then tap **Start chatting**.
- **Chat** (`app/chat.js`): uses the chosen color and sets the header title to the user's name.

## Tech
Expo SDK 54 • React Native • expo-router

## Run locally
```bash
npm install
npm start
# press i for iOS simulator (or a for Android)
